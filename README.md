#### Instructions:
##### Clone the Project:
```bash
git clone https://github.com/ibuybooks/william-blake.git
cd william-blake
```
##### Compile the Document:
```bash
lualatex WilliamBlake
```

##### Process the Bibliography:
```bash
biber WilliamBlake
```

##### Compile with the Bibiography:
```bash
lualatex WilliamBlake
```
